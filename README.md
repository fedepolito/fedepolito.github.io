Tecnologías Utilizadas
HTML5: Estructura semántica y organización del contenido web.

CSS3: Diseño visual, gestión de tipografías y desarrollo de interfaz adaptable (Responsive Design).

JavaScript: Implementación de interactividad, control del menú de navegación y efectos dinámicos de desplazamiento.

Font Awesome: Integración de iconografía para redes sociales y elementos de interfaz.

Google Fonts: Optimización de la legibilidad mediante la gestión de fuentes tipográficas.
